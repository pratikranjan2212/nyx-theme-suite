# Change Log

All notable changes to the "true-dark" extension will be documented in this file.

## [3.0.0] - 2026-05-20

### Added
- Ported the full "True Dark" IntelliJ theme suite (v3.0.0) to VS Code.
- Added **True Dark** & **True Dark Islands** (refined `#121212` backgrounds).
- Added **True Black** & **True Black Islands** (OLED-friendly `#000000` pitch black backgrounds).
- Added **True Light** & **True Light Islands** (premium cool light blue-white backgrounds).
- Styled all editor syntax token colors matching the original IntelliJ specifications.
